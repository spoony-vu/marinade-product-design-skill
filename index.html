Claude Code Skill

Marinade Product Design

A design system skill for Claude Code that enforces Marinade Finance's visual language, color tokens, and component conventions across every tool you build.

~/.claude/skills/marinade-product-design/
Setup
How to Install

Pick one of these two methods, then start building.

A

Install via npm recommended

One command to install the skill globally:

npx claude-code skill install marinade-product-design

Or from the Git repo directly:

npx claude-code skill install github:spoony-vu/marinade-product-design-skill
B

Clone manually

Clone the repo and copy the skill folder:

git clone https://github.com/spoony-vu/marinade-product-design-skill.git cp -r marinade-product-design-skill/skill ~/.claude/skills/marinade-product-design

The skill folder contains:

SKILL.md # Design system rules + token reference tokens.json # Clean token file (light + dark)

Then start building

The skill auto-activates when Claude detects Marinade tool work. It asks 4 discovery questions before scaffolding, then generates a full project with tokens wired in.

# Just describe what you need: "Build me a validator dashboard for Marinade" # The skill enforces: # - Token-based colors (never hardcoded hex) # - Dark mode default + light mode support # - Shadcn UI + Tailwind + Geist + Lucide icons # - Recharts with interactive tooltips # - Solana data formatting patterns # - Auto-fixes UI quality issues

Usage
Example Prompts

The skill activates when you ask Claude Code to build Marinade tools. Here are prompts that trigger it.

"Build a validator performance dashboard"
Creates a Next.js app with stat cards, validator table, commission chart, and dark mode toggle. All using Marinade tokens.
"Add a staking rewards calculator page"
Extends existing project with a new route, input form for SOL amount, and projected returns using Geist Mono for all numbers.
"Create an alert management tool for monitoring validators"
Uses destructive/warning/info tokens for status badges, tag colors for categories, and sidebar navigation.
"Style this existing React component to match Marinade's design"
Rewires existing component to use CSS variables, replaces hardcoded colors, adds proper dark/light mode support.
